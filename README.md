# Tide

Tide is a local-first period tracking app built as a Progressive Web App.

Users can log period days, view cycle estimates, check reminders, and export or import their data with a simple JSON backup.

## Features

- Log period days
- View cycle and phase estimates
- Check reminders and snooze them
- Switch between multiple home screen views
- Keep data stored locally in the browser

## Stack

- React
- TypeScript
- Vite
- React Router
- PWA support with `vite-plugin-pwa`
- Vitest, Playwright, and Biome

## Install

```bash
pnpm install
pnpm dev
```
